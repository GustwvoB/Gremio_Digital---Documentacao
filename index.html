Uma solução para a Educação

Sobre o Projeto


Desenvolvido como projeto integrador, este trabalho propõe uma abordagem inovadora para um problema real da área educacional. Toda a documentação técnica e administrativa está centralizada aqui.


O projeto é orientado pela professora Kadidja Valeria Reginaldo de Oliveira e desenvolvido na instituição UniCEUB.
